# pruebase2ePruebasAutomatizas

## How to run
Para correr las pruebas con Cypress usar:
```
npm install -g cypress
cypress run
```


#####FUNCIONALIDADES A TRABAJAR#################

- Crear Post
- Publicar Post
- Listar Posts programados
- Listar todos los posts
- Crear página
- Listar páginas
- Eliminar página
- Crear miembro
- Listar miembros
- Eliminar miembro
- Crear tag
- Listar Tags
- Eliminar tag
- Importar usuarios
- Admin de Ghost
- Autenticación
- Actualizar configuración general
